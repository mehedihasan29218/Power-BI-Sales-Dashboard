# 📊 Power BI Sales Dashboard

This project showcases a Power BI Sales Dashboard designed with a focus on clarity, usability, and business-driven insights.

## 🔍 Overview

The dashboard presents key business metrics including:
- Sales performance
- Profit trends
- Product ranking (Top N analysis)
- Regional insights

Two design approaches are explored:
- 🌞 Light Theme → optimized for readability
- 🌙 Dark Theme → optimized for visual impact

## ⚙️ Features

- KPI-driven executive summary  
- Interactive filters (Year, Month, Quarter)  
- Top/Bottom product ranking  
- Profit margin analysis  
- Clean and structured layout  

## 🛠️ Tools & Technologies

- Power BI  
- DAX (Data Analysis Expressions)  
- Power Query  

## 📂 Dataset

This dashboard is built using a **public demo dataset** for demonstration purposes.  
No real or confidential data is used.

## 🎯 Purpose

This project demonstrates:
- Dashboard design principles  
- Business storytelling with data  
- UI/UX improvements (Light vs Dark comparison)
